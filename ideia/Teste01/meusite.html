Configurando o Git Bash no PC

1º Passo: Registrar seu GitHub no Git Bash

1- Tenha o Git Bash instalado e aberto em seu computador;
2- Pesquise no google 'configuração inicial git' / provável: git.scm;
3- Copie os seguintes codigos em ordem e cole no seu Git Bash:

- Primeiro adicione: ( $ git config --global user.name "Fulano de Tal" ), no seu git bash, trocando o username pelo seu nome do github e de enter
- No proximo adicione: ( $ git config --global user.email fulanodetal@exemplo.br ), trocando agora seu email utilizado na conta do github e de enter

2º Passo: Clonando seu arquivo do GitHub

1- Entre no seu GitHub, e vá em seu repositório e clica na pasta que deseja clonar;
2- Após isso, clique no code, aparecerá uma mensagem, escolha em https e clica em copiar;
3- Com o link, vá no git bash e escreva "git clone + link" e de enter, que os arquivos serão clonados para sua área de trabalho.
